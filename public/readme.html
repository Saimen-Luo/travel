1. 浏览方式
    1. 使用手机浏览器打开链接即可正常浏览
    2. 使用电脑 Chrome 浏览器打开链接，按 F12 打开开发者工具，再按 Ctrl + Shift + M 打开手机模拟器，然后刷新页面进行浏览（如果模拟器显示异常，请切换模拟机型）
  2. 首页 banner 图片轮播，可左右循环滑动
  3. 下面 icons 分类轮播，可左右滑动，不能循环
  4. 点击右上角城市名称可进入城市选择页面
    1. 点击搜索框，输入文字或拼音可搜索城市
    2. 点击右侧字母表字母如D，城市列表会自动滚动到字母D，并弹窗提示当前字母
    3. 滑动右侧字母表，城市列表会自动滚动到当前滑到的字母，并弹窗提示当前字母
    4. 点击任一城市，设置为当前城市，并跳转回首页，首页城市同步改变
    5. 点击左上角箭头亦可返回首页
  5. 点击"热门推荐"任一景点，即可进入景点详情页
    1. 点击图片可进入图库组件，可轮播浏览图片
    2. 点击图库黑色区域即可退出
    3. 图库进入和退出有淡入淡出过渡效果
    4. 门票列表，使用递归组件完成
    5. 详情页面顶部为黑色箭头退出按钮；当页面向下滚动时，切换为固定的头部组件，并且有从虚到实缓慢出现的效果（上滑也会逐渐透明消失）；点击箭头亦可返回首页
